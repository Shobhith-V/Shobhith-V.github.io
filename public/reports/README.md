# Reports Directory

Place PDF reports and publications in this directory.

## Usage

1. Add your PDF files here (e.g., `research-report-2024.pdf`)
2. Reference them in `data/projects.json`:
   ```json
   "pdf": "public/reports/research-report-2024.pdf"
   ```
3. Or add them to `data/publications.json` for the publications page

PDFs will be viewable inline in the browser and available for download.
