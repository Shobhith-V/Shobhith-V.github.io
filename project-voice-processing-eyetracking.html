Cognitive Processing of Voice Constructions

Research Project • December 2024 – Present

Psycholinguistics Cognitive Neuroscience Statistical Modeling
Code • PDF

This project investigates how syntactic voice constructions (active vs. passive) influence real-time sentence processing and attentional allocation. I designed and conducted a psycholinguistic eye-tracking experiment using PsychoPy on the EyeLink Portable Duo system to capture gaze dynamics during sentence comprehension.

To analyze time-resolved eye-movement data from 48 participants, I employed Generalized Additive Mixed Models (GAMMs), enabling flexible modeling of nonlinear gaze trajectories while accounting for subject-level variability. The analysis aims to uncover cognitive load differences and validate the agent-first hypothesis in incremental sentence parsing.

Conceptually, this work formalizes voice alternations as representational transformations and examines how syntactic role assignment modulates attentional dynamics during language comprehension.
