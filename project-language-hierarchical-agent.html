Language-Based Hierarchical Agent for Reasoning

Course Project • July 2024 – November 2024

Reinforcement Learning Representation Learning Symbolic AI
Code • PDF

This project explored the use of language as a representational substrate for hierarchical reasoning in partially observable environments. I encoded image-based POMDP environments into structured language representations to support reasoning and action selection.

I developed an autoencoder-like architecture that reconstructs images from text representations, enabling feature extraction for a downstream PPO agent. Through systematic analysis, I identified failure modes arising from limited scene context extraction, linking these shortcomings to language model capacity, autoencoder complexity, and the constraints of text-based representations.
