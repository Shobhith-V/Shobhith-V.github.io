Overview

Predictive coding as a unified theory of brain computation

This project explored the predictive coding paradigm as a computational framework for information processing in the brain — a theory proposing that the brain continuously generates predictions and updates them based on prediction errors, unifying perception, inference, and learning under a single principle.

What I Did

From theory to implementation

I conducted an extensive literature review covering theoretical foundations, neural interpretations, and modern ML formulations of predictive coding. To develop concrete intuition, I implemented a predictive coding model from scratch and replicated its learning dynamics on MNIST.

  • Deep review of Rao & Ballard (1999), Friston's Free Energy Principle, and PC-learning variants
  • Implemented error-driven inference with hierarchical representations and local learning rules
  • Benchmarked against standard backpropagation on MNIST classification
  • Analyzed forward model structure and efference copy in cerebellar models
  • Formalized predictive coding as variational inference over generative models

The implementation emphasized biological plausibility — local learning rules, lateral inhibition, and error unit architectures that map to known cortical microcircuit motifs.

Key Insights

Bridging neuroscience and machine learning

Through this work, I gained practical insight into how predictive coding unifies perception, inference, and learning — and how biologically inspired principles can be translated into executable computational models that rival backpropagation in certain settings while remaining far more interpretable.
