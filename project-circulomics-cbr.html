Circulomics: A Case-Based Reasoning Framework for Circularity

Course Project • July 2025 – November 2025

Symbolic AI Explainable AI Cognitive Neuroscience
PDF

This project addressed the problem of circularity in AI and cognitive systems by developing a Case-Based Reasoning (CBR) framework to detect and resolve feedback-driven failure modes. Circular dependencies were represented as structured cases capturing loop topology and system dynamics.

I designed a feature schema and topology-aware similarity metric to retrieve analogous loop structures across domains, such as mapping the Barber Paradox to instability in recurrent neural networks. Solutions were adapted using a guided LLM-based reasoning procedure combined with symbolic validation, enabling reliable cross-domain transfer of corrective strategies.
