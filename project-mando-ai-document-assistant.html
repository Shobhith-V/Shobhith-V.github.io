Mando AI – Multimodal Document Q&A Assistant

Hackathon (Finalist) • February 2025 – March 2025

Machine Learning Multimodal Modeling Scientific Computing Human–Machine Interaction
Code

This project involved building a Streamlit-based AI assistant with a conversational interface for question answering over heterogeneous document formats using LLM-powered semantic search. The system was designed to support interactive exploration of unstructured, semi-structured, and structured data within a unified chat interface.

I engineered a robust data ingestion pipeline that performs OCR to extract text from images, crawls web pages and embedded links to augment document context, and parses structured data files (CSV, Excel, JSON) for downstream analysis. For structured inputs, the system dynamically generated interactive Plotly visualizations to support exploratory data analysis.

Additionally, I enabled secure execution of LLM-generated Python code to perform custom data analysis directly within the application. The chat interface was designed with session-level history and context management, enabling multi-turn, stateful interactions. This project was selected as a finalist submission at the Mando AI hackathon.
