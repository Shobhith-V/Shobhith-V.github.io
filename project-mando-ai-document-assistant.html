Overview

A unified AI interface for heterogeneous document formats

Mando AI is a Streamlit-based conversational assistant for question answering over heterogeneous document formats — PDFs, images, spreadsheets, web pages, and more — using LLM-powered semantic search and RAG. The system was designed for interactive, stateful exploration of unstructured data through a natural chat interface.

Data Ingestion Pipeline

OCR, web crawling, and structured data parsing

I engineered a robust data ingestion pipeline that handles multiple document modalities:

  • OCR: Extract text from images and scanned PDFs using Tesseract + layout analysis
  • Web crawling: Crawl pages and embedded links to augment document context automatically
  • Structured data: Parse CSV, Excel, and JSON for downstream analysis and visualization
  • Dynamic Plotly charts: Auto-generated interactive visualizations for structured inputs
  • Code execution: Secure sandboxed execution of LLM-generated Python for custom analysis
Interface Design

Stateful multi-turn conversations over documents

The chat interface was designed with session-level history and context management, enabling multi-turn, stateful interactions where users can ask follow-up questions, refine queries, and explore documents conversationally. Context windows were managed to balance retrieval breadth with LLM token limits.

The entire system was deployed on DataIku, making it accessible to non-technical lab scientists without any setup or environment management — a key design constraint from the hackathon brief.
