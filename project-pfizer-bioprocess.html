Overview

ML-accelerated chromatography process development

An eight-month internship at Pfizer's Bioprocess R&D downstream modeling team, focused on building computational tools and machine learning methods to accelerate chromatography process development. Two primary projects: a simulation GUI and a CNN-based active learning workflow for parameter estimation.

Project 1: Simulation GUI

Interactive chromatography simulation platform

I designed and deployed an interactive application enabling laboratory scientists to configure, run, and analyze chromatography simulations without writing code — dramatically lowering the barrier to model-driven experimentation.

  • Streamlit for the user interface — accessible from any browser, no installation
  • CADET as the simulation backend for mechanistic chromatography modeling
  • DataIku for deployment and access management within Pfizer's infrastructure
  • Parameter sweep, result visualization, and export built into the interface
Project 2: Parameter Estimation

CNN + active learning for HIC isotherm fitting

I designed and implemented a CNN-based active learning workflow for parameter fitting in Hydrophobic Interaction Chromatography (HIC) isotherm models. The approach reduced parameter fitting time while improving estimation accuracy by approximately 5%.

  • CNN surrogate model trained on CADET-generated simulation data
  • Active learning loop selects most informative experiments to run next
  • Generalized to support multiple isotherm formulations (Langmuir, SMA, etc.)
  • Validated against experimental data from two production-ready molecules
